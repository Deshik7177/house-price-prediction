# House Price Prediction App

This project predicts house prices using a machine learning model trained on the provided dataset. It features a modern, professional UI built with Streamlit.

## Features
- Data preprocessing and model training
- Interactive UI for user input and predictions
- Clean, professional design

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the app:
   ```bash
   streamlit run app.py
   ```

## Project Structure
- `app/app.py`: Main Streamlit app
- `app/model.py`: Model training and prediction logic
- `app/utils.py`: Data preprocessing utilities
- `House Price Prediction Dataset.csv`: Dataset
